### Windows

1. Execute setup.ps1

2. Uncomment below code in msys2 batfile

  ```
set MSYS=winsymlinks:nativestrict
  ```

3. Excute setup.sh from msys2 shell (Administrator)

4. Update submodule

  ```
$ git submodule update --init
  ```

### Mac, Linux

1. Execute setup.sh

2. Update submodule

  ```
$ git submodule update --init
  ```
